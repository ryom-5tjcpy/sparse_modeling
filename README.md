このリポジトリはPythonでスパースモデリングのサンプルプログラムをGeminiで生成したものである。

[sample.py](sample.py)はスパースモデリングのサンプルプログラムである。
このプログラムでは、まずランダムなデータを生成し、生成したデータにノイズを多数追加したのちに
線形回帰とLASSO回帰の2つのモデルを適用している。

[sparse_paint.py](sparse_paint.py)はスパースモデリングを使用した画像修復のサンプルプログラムである。
このプログラムでは、初めにテスト画像を作成する。
テスト画像はシンプルなグラデーション画像である。
次に、作成したテスト画像の50%の画素を欠損させる。
欠損した画像をスパースモデリングによって修復させる。
このとき、計算をシンプルにするために辞書学習によって画像のパッチごとのスパース表現を求めている。
ただし、使用した辞書はランダム規定であるため修復の精度は低い。
